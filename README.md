# Classification-with-Logistic-Regression
AI/ML Internship Task 4 – Classification using Logistic Regression with model evaluation using confusion matrix, precision, recall, and ROC-AUC.

Dataset Used:-
Breast Cancer Wisconsin Dataset
Used to classify tumors as:
Malignant (Cancerous)
Benign (Non-cancerous)
Tools Used:-
Python
Pandas
Matplotlib
Scikit-learn
Tasks Performed:-
Imported and preprocessed the dataset
Split data into training and testing sets
Standardized features using StandardScaler
Trained a Logistic Regression model
Evaluated model using:
Confusion Matrix
Precision
Recall
ROC-AUC Score
Understood threshold tuning and sigmoid function

Learning Outcome:-

Through this task, I learned:

Binary classification
Logistic Regression
Model evaluation metrics
Classification performance analysis
